# КотоВечер 🌙

**CatEvening** — сайт для вечернего наслаждения котиками. Тёмная, ночная эстетика с анимацией звёзд, фазами луны и случайными котами.

## Возможности

- 🌙 **Тёмная тема** по умолчанию с переключателем на светлую
- 🐱 **Случайные фото котов** через [TheCatAPI](https://thecatapi.com)
- 📖 **Случайные факты о котах** через [catfact.ninja](https://catfact.ninja/fact)
- 🕐 **Текущее время** с приветствием (утро/день/вечер/ночь)
- 🌓 **Фаза луны** на основе дня месяца
- ✨ **Анимированное звёздное небо** и созвездия на фоне
- 🐾 **Кнопка «Муркни!»** со счётчиком и эффектом искр
- 📱 **Адаптивный дизайн** для мобильных устройств
- ⌛ **Скелетон-загрузка** пока контент подгружается

## Использование

Просто откройте `index.html` в браузере. Нажмите «Муркни!» чтобы получить нового котика и факт.

Данные кешируются в `localStorage`: тема, счётчик мурчалок.

## Технологии

- Чистый HTML + CSS + JavaScript (без библиотек)
- Google Fonts: Inter + JetBrains Mono
- TheCatAPI v1
- catfact.ninja API

## Файлы

- `index.html` — полностью самодостаточная страница (всё в одном файле)

## Отличия от КотоДня

КотоВечер — ночная версия с другим настроением:
- Тёмно-фиолетовая цветовая гамма вместо тёплой дневной
- Звёзды вместо размытых орбов на фоне
- Луна вместо кошачьей мордочки как логотип
- Фазы луны и приветствие по времени суток
- «Муркни!» вместо «Мяу!»
- Эффект искр при нажатии
